# Tell Me More - Podcast Playlist App

## Project Overview

**Mission:** Build a personalized podcast playlist app that discovers and recommends individual podcast episodes based on user-defined topics of interest.

**Core Value Proposition:** Instead of following podcast shows, users follow *topics* and get a daily playlist of the most relevant individual episodes across all podcasts.

---

## Key Features

### MVP (Web App)
1. User registration & authentication
2. Topic preference management (plain text)
3. Daily personalized playlist of relevant episodes
4. Relevance scoring + date-based sorting
5. Podcast catalog ingestion (Apple Podcasts API)

### MVP2 (iOS App)
1. Native iPhone experience
2. Push notifications for new relevant episodes
3. Offline episode caching
4. Audio player integration
